# API Counter
Um projeto feito para estudar mais sobre o uso de backend com ASP.NET e frontend com React + Typescript + Tailwind.

# Sobre o projeto
A ideia do projeto foi para me atualizar mais sobre os hooks e atualizações do React, novidades do Tailwind etc.
Ele é divido em:
- Backend: ASP.NET simples, que responde uma rota de incrementação do valor recebido. Dividido em modelo de features.
- Frontend: Inicialmente React + Vite básico com Typescript, e para estudos e melhor interface foi adicionado o Tailwind. Atualmente usando fetch e não axios.
